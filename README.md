# Gráfico de Barras com Python e Matplotlib: Visualização de Medalhas nas Olimpíadas de 2020

Este é um projeto simples em Python que usa a biblioteca Matplotlib para criar um gráfico de barras que mostra o número de medalhas ganhas por alguns países nas Olimpíadas de 2020.

## Pré-requisitos

- Python 3.x
- Biblioteca Matplotlib

## Como usar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `medalhas.py` em seu editor de código Python.
3. Certifique-se de que a biblioteca Matplotlib esteja instalada. Caso contrário, instale com o comando `pip install matplotlib` no terminal ou prompt de comando.
4. Execute o código.
5. O gráfico de barras será exibido na tela.
